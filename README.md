# 🎯 Quiz Game Web App

A simple and interactive **Quiz Game** built using **HTML, CSS, and JavaScript**.  
This project is perfect for beginners to understand DOM manipulation, event handling, and basic game logic.

---

## 🚀 Features

- 🟢 Start screen to begin the quiz  
- ❓ Multiple-choice questions  
- 📊 Live progress bar  
- 🧮 Real-time score tracking  
- ✅ Instant feedback (correct / incorrect answers)  
- 📈 Final result with performance message  
- 🔄 Restart quiz option  
- 🎨 Clean and modern UI (gold + cream theme)

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## 🎮 How to Play

1. Click **Start Quiz**
2. Select the correct answer
3. Score updates automatically
4. See final result at the end
5. Click **Restart Quiz** to play again

---

## 📊 Scoring System

- 100% → Excellent 🌟  
- 80%+ → Great 👍  
- 50%+ → Good 🙂  
- Below 50% → Try Again 💪
